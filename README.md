# HW2

Консольное приложение для управления рецептами: создание блюд, добавление ингредиентов, масштабирование порций и генерация списка покупок.

# Структура проекта

- `Ingredient.py` - класс ингредиента
- `Recipe.py` - класс рецепта
- `ShoppingList.py` - класс списка покупок
- `DietaryRecipe.py` - класс диетического рецепта

# Установка

```bash
git clone https://github.com/ElenaRussu/HW2.git
cd HW2
pip install -r requirements.txt
```

# Использование

Запуск тестов:
```bash
cd main
PYTHONPATH=. pytest test/ -v
```
